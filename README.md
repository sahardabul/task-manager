## מנהל משימות
## תיאור קצר של האפליקציה

אפליקציית מנהל משימות לניהול משימות יומיומיות.
 האפליקציה מאפשרת הוספה, עריכה, מחיקה וסימון משימות כהושלמו/לא הושלמו, כולל סינון לפי סטטוס ושמירה אוטומטית ב־localStorage כך שהנתונים לא נעלמים ברענון.

## פיצרים עיקריים

הוספת משימה חדשה

סימון משימה כהושלמה / ביטול סימון

עריכת טקסט משימה

מחיקת משימה

סינון: הכל / פעילות / הושלמו (עם סימון לפילטר הפעיל)

שמירה ושחזור אוטומטיים ב־localStorage

כפתור "נקה משימות שהושלמו" (מחיקה בבת אחת של כל המשימות שהושלמו)

כפתור "סמן הכל כהושלם" (סימון בבת אחת של כל המשימות)

## הוראות להרצת הפרויקט מקומית

דרישות: Node.js + npm

התקנת תלויות:
npm install

הרצת הפרויקט:
npm run dev

## קומפוננטות 

App – מנהל state ראשי (tasks/filter), מפעיל CRUD, סינון, ושמירה/טעינה מ־localStorage.

TaskForm – טופס להזנת משימה חדשה והפעלת הוספה לרשימה.

FilterBar – בחירת פילטר (הכל/פעילות/הושלמו) עם סימון ויזואלי לפילטר הנבחר.

TaskList – מציג את רשימת המשימות בהתאם לפילטר שנבחר.

TaskItem – מציג משימה בודדת ומאפשר סימון הושלמה/לא הושלמה, עריכה ומחיקה.

## מגבלות / באגים ידועים

אין סנכרון בזמן אמת בין טאבים שונים (שינויים ב־localStorage בטאב אחד לא מתעדכנים אוטומטית בטאב אחר).

עריכת משימה לא מאפשרת טקסט ריק.

הנתונים נשמרים רק ב־localStorage של הדפדפן המקומי; מעבר מחשב/דפדפן או מחיקת נתוני גלישה ימחקו את המשימות.
